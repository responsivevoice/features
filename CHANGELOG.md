# @responsivevoice/features

## 2.0.0

First public release of the rebuilt ResponsiveVoice — a complete, TypeScript-first rewrite of the original library, now split into focused, independently-versioned packages.

`@responsivevoice/features` is a collection of drop-in accessibility and engagement plugins for sites using ResponsiveVoice.

### Highlights

- Inline web player with playback controls and paragraph navigation
- Welcome, inactivity, end-of-page, and exit-intent spoken messages
- Speak-on-select and speak-links-on-hover
- Keyboard paragraph and tab navigation with speech

Documentation: https://docs.responsivevoice.org
